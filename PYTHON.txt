No Python, caso queira escrever algo na tela, você coloca a mensagem entre aspas simples ou duplas, logo em seguida os parênteses, e antes de tudo o print. Mas para efetuar contas você não usa as aspas, se usar, irá aparecer a conta em vez o resultado.

EX:
print('Olá, Mundo!') #irá escrever exatamente oque está escrito
print('7+3') #aqui em vez ed mostrar o resultado vai aparecer exatamente oque está escrito
#resultado: 7+3
print(7+3) #aqui vai mostrar o exato resultado do calculo
#resultado: 10

Isso acontece porque toda escrita dentro e aspas é uma "string", da mesma forma que é no js, então ao tirar as aspas automaticamente se torna um number. Outro exemplo bem interessante é colocando cada valor em suas próprias aspas.

EX:

#resultado: 74

Mas existem horas que em vez do '+', será usada a',', não têm uma hora exata para usar, mas com o tempo você irá se acostumar.

Aqui está uma lista de todos os "is"(métodos de strings):
1.isalnum()
Retorna True se todos os caracteres da string são alfanuméricos (letras ou números), e a string não está vazia.

2.isalpha()
Retorna True se todos os caracteres da string são alfabéticos (letras) e a string não está vazia.

3.isascii()
Retorna True se todos os caracteres da string são caracteres ASCII (valores entre 0 e 127).

4.isdecimal()
Retorna True se todos os caracteres na string forem dígitos decimais (0-9).

5.isdigit()
Retorna True se todos os caracteres na string forem dígitos (0-9), ou caracteres numéricos como superescritos e subscritos.

6.isnumeric()
Retorna True se todos os caracteres da string forem numéricos (incluindo caracteres como numerais romanos e números fracionados).

7.islower()
Retorna True se todos os caracteres da string estiverem em minúsculas.

8.isprintable()
Retorna True se todos os caracteres da string forem imprimíveis (não incluem caracteres de controle como novas linhas ou tabulações).

9.isidentifier():
Verifica se a string é um identificador válido (como um nome de variável em Python).

10.isspace():
Verifica se todos os caracteres da string são espaços em branco (incluindo espaços, tabs e quebras de linha).

11.istitle():
Verifica se a string está no formato de título, ou seja, se cada palavra começa com uma letra maiúscula e o restante é minúsculo.

12.isupper():
Verifica se todos os caracteres da string estão em maiúsculas.

Operadores Aritméticos

Com o .fotmat é possível fazer alinhamentos de palavras dentro de espaços definidos.

EX:

#resultado: Prazer em te conhecer,         nome        !

print('Prazer em te conhecer, {:=^20}!'.format(nome)) #aqui você colocou nome em 20 espaços de forma centralizada, e colocou "=" nos espaços vazios

Os loops são uteis para realizar tarefas repetitivas. Existem 2 tipos de estruturas de repetição:

Para abrir um arquivo no python é usado uma variavel atribuida a função open()
Nessa função atribuimos dois parametrso, o primeiro sendo o caminho(nome do arquivo) e o segundo é a maneira que vamos abrir ele.
Existem 5 maneiras de abrir o arquivo:
'r'	Leitura (padrão). O arquivo deve existir.
'w'	Escrita. Cria um arquivo novo ou sobrescreve o existente.
'a'	Acrescentar. Adiciona ao final sem sobrescrever.
'b'	Binário. Usado com 'r', 'w', ou 'a' para arquivos binários.
'x'	Criação exclusiva. Retorna erro se o arquivo já existir.

Para ler arquivos, precisamos abrir ele primeiro. Após isso temos diversas maneiras de ler, sendo elas:
read() Lê o conteúdo completo do arquivo como uma string
readline() Lê apenas uma linha do arquivo por vez
readlines() Retorna uma lista onde cada linha do arquivo é um item

Existem também as escritas em arquivos no python que também é necessário estar com o arquivo aberto, as maneiras de escrita são:
write() Escreve uma string no arquivo
writelines() Escreve uma lista de strings no arquivo
Importante lembrar que o metodo de abertura deve estar relacionado com oquer vai fazer(ler o arquivo, escrever) e que é necessário fechar os arquivos após manipula-los para liberar recursos do sistema.
